# 스킬 숙련도 시스템 규칙

## 1. 목표

캐릭터별 대표 액티브 스킬과 고유 패시브를 장기적으로 성장시키는 시스템이다.

- 영구 장비, 던전 런 증강과 별개의 캐릭터 성장 축으로 운영한다.
- PvP와 PvE는 같은 스킬 규칙과 같은 숙련도 효과를 사용한다.
- 캐릭터의 고유 스킬 정체성은 유지하고, 숙련도로 수치와 각성을 확장한다.

## 2. 기본 구조

| 항목 | 규칙 |
|---|---|
| 숙련도 대상 | 캐릭터별 대표 액티브 1개 + 고유 패시브 1개 |
| 숙련도 형태 | 액티브와 패시브가 하나의 공용 숙련도를 공유 |
| 시작 레벨 | Lv.1 |
| 최대 레벨 | Lv.50 |
| 성장 재화 | 캐릭터 귀속 스킬 성장 포인트 |
| 투자 방식 | 포인트 1개 투자마다 숙련 XP 1 증가 |
| 레벨업 | 다음 레벨 필요 숙련 XP를 모두 채우면 즉시 1레벨 상승 |
| 최대 레벨 | Lv.50 달성 후 투자 불가 |

스킬 성장 포인트는 플레이어 계정 레벨의 SP, 영구 장비, 던전 런 증강과 서로 다른 재화·상태로 관리한다.

## 3. 숙련 XP 요구량

다음 레벨에 필요한 숙련 XP는 현재 숙련도를 기준으로 계산한다.

```text
필요 숙련 XP = ceil(현재 숙련도 / 5)
```

| 현재 숙련도 | 다음 레벨 필요 투자 횟수 |
|---|---:|
| Lv.1 ~ Lv.5 | 1회 |
| Lv.6 ~ Lv.10 | 2회 |
| Lv.11 ~ Lv.15 | 3회 |
| Lv.16 ~ Lv.20 | 4회 |
| Lv.21 ~ Lv.25 | 5회 |
| Lv.26 ~ Lv.30 | 6회 |
| Lv.31 ~ Lv.35 | 7회 |
| Lv.36 ~ Lv.40 | 8회 |
| Lv.41 ~ Lv.45 | 9회 |
| Lv.46 ~ Lv.49 | 10회 |

예시:

```text
마천결계 숙련도 Lv.12 / 50
숙련 XP 2 / 3
보유 스킬 성장 포인트 7

[ +1 투자 ]
```

위 상태에서 `+1 투자`를 누르면 숙련 XP가 `3 / 3`이 되고, 숙련도가 Lv.13으로 상승한다.

## 4. 성장 보상

### 매 레벨 기본 성장

숙련도가 1레벨 오를 때마다 액티브와 패시브의 수치가 함께 소폭 상승한다.

- 액티브: 피해, 반경, 지속 시간, 둔화, 충전 효율 등 캐릭터별 수치
- 패시브: 스택 유지 시간, 보조 피해, 보호막, 회복, 발동 조건 등 캐릭터별 수치
- 수치의 실제 증가량은 캐릭터별 스킬 정의에서 관리한다.

### 각성 구간

| 레벨 | 보상 유형 | 설명 |
|---|---|---|
| Lv.5 / 10 / 15 | 소각성 | 기존 효과를 보강하는 작은 추가 효과 |
| Lv.20 / 30 / 40 | 대각성 | 플레이 방식에 영향을 주는 큰 강화 |
| Lv.25 / 35 / 45 | 빌드 분기 | 추후 2개 중 하나를 선택하는 전용 강화 슬롯 |
| Lv.50 | 최종 각성 | 캐릭터의 완성형 고유 효과 |

빌드 분기 규칙은 초기 출시 시 비활성화할 수 있다. 이 경우 해당 레벨은 추가 수치 성장과 사전 고지된 고정 효과만 제공한다.

## 5. 스킬 성장 포인트 획득

포인트는 획득한 캐릭터에게 귀속된다. 예를 들어 찬휘로 획득한 포인트는 찬휘의 숙련도에만 투자할 수 있다.

| 콘텐츠 | 포인트 지급 기준 |
|---|---|
| PvP | 경기 완료 보상 |
| 스테이지 던전 | 스테이지 클리어 보상 |
| 무한 생존 | 완료 웨이브 정산 보상 |
| 이벤트·업적 | 캐릭터 지정 보상 또는 선택 보상 |

정확한 지급량은 콘텐츠별 보상 테이블을 만들 때 별도로 확정한다.

## 6. UI 규칙

성장 탭의 캐릭터 스킬 영역에는 다음 정보를 표시한다.

```text
마천결계 · 숙련도 Lv.12 / 50
████████████████░░░░  2 / 3

보유 스킬 성장 포인트: 7
[ +1 투자 ]  [ +3 빠른 투자 ]

다음 레벨: 결계 틱 피해 +1
다음 각성: Lv.15 · 둔화 강화
```

- `+1 투자`: 포인트 1개를 사용해 숙련 XP 1을 투자한다.
- `+3 빠른 투자`: 보유 포인트와 다음 레벨 필요량 중 가능한 만큼 최대 3개를 투자한다.
- 최대 레벨에서는 투자 버튼을 비활성화하고 `최종 각성 완료`를 표시한다.
- 각성 효과는 잠금 상태, 다음 해금, 해금 완료 상태를 구분해 표시한다.

## 7. 구현 원칙

- 숙련도와 포인트는 Convex 서버 데이터가 원본이다.
- 클라이언트는 서버 구독값을 표시하며, 레벨과 포인트를 로컬에서 확정하지 않는다.
- 투자 요청은 서버 mutation에서 포인트 보유량, 최대 레벨, 필요 XP를 검증한다.
- 캐릭터별 수치와 각성 효과는 해당 캐릭터 파일에 둔다.
- 공통 숙련도 계산, 포인트 검증, 보상 지급은 공용 서버·헬퍼 로직으로 관리한다.
- 던전 런 증강은 해당 런에만 적용되며 영구 숙련도를 변경하지 않는다.

## 8. 찬휘 예시 (초안)

| 숙련도 | 액티브: 마천결계 | 패시브: 마기 |
|---|---|---|
| Lv.1 | 결계 틱 피해 기본값 | 피해 시 마기 1스택 |
| Lv.5 | 결계 반경 소폭 증가 | 마기 폭발 반경 소폭 증가 |
| Lv.10 | 결계 지속 시간 증가 | 마기 폭발 피해 증가 |
| Lv.15 | 결계 둔화 강화 | 마기 유지 시간 증가 |
| Lv.20 | 결계 시전 보호막 | 폭발 적중 시 보호막 |
| Lv.25~45 | 추후 빌드 분기 | 추후 빌드 분기 |
| Lv.50 | 천마 현신 최종 각성 | 천마 현신 최종 각성 |

찬휘의 정확한 수치와 Lv.25~50 세부 각성은 별도 캐릭터 설계서에서 확정한다.
