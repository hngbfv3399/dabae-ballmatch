## 제품 원칙

- PvE 던전은 매 런 증강과 아이템으로 다른 빌드를 만드는 공간이다.
- 캐릭터 레벨, 스킨, 경험치 포인트 아이템, 해금 스킬은 영구 유지한다.
- Lv.1~30은 스킬 해금·증가 구간이며, Lv.31부터는 해금된 스킬을 강화하는 숙련 구간으로 전환한다.
- 영구 성장으로 스킬 쿨타임·충전 속도를 낮추지 않는다. 스킬 강화는 유지시간, 추가 시전, 효과 범위·횟수처럼 행동 자체를 바꾼다.
- 영구 레벨 성장에는 HP·공격력과 함께 정수형 DEF 보호막을 포함하며, 기본 DEF와 스탯 성장 레벨의 +1 DEF가 HP보다 먼저 피해를 흡수한다.
- 런 전용 증강과 아이템은 던전 종료 시 폐기한다.
- 일반 PvP는 공정성을 위해 런 전용 효과를 적용하지 않는다.
- 전용 캐릭터 효과는 `CharacterConfig` 훅으로 연결하며 공용 엔진에 캐릭터 ID 분기를 추가하지 않는다.
