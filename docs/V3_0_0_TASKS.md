# v3.0.0 구현 작업 목록

## 확정된 제품 규칙

- 기존 보스전은 삭제하고, 첫 PvE 던전 `초원의 슬라임 소굴`을 메인 콘텐츠로 추가한다.
- 캐릭터 레벨과 경험치는 서버 공용이다. 어떤 클라이언트가 캐릭터로 던전을 클리어해도 같은 캐릭터의 레벨이 오른다.
- 레벨은 1부터 30까지이며, PvE와 PvP에 공통으로 적용한다. 추가 장착 스킬은 이번 버전에서 만들지 않는다.
- 첫 던전은 5스테이지이며, 전 스테이지의 적을 처치해야 보상을 받는다. 실패 보상은 없다.
- 첫 클리어는 450 XP, 이후 클리어는 200 XP를 지급한다.
- 가챠는 KST 자정마다 기본 5회를 지급하고, 던전 클리어 3회마다 추가 1회를 지급한다. 추가 횟수 상한은 없다.
- 가챠는 전 서버 공용 공통 스킨 24종으로 시작한다. 새 스킨은 전역 도감에 해금되고, 중복은 선택 캐릭터에게 등급별 XP를 지급한다.
- PvP 랭킹은 개인전·팀전·토너먼트를 분리하고, 승률과 점수를 함께 표시한다.
- 8주 리셋에서 레벨·누적 XP·공용 스킨 도감만 유지한다.
